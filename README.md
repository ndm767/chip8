# Chip8

A Chip8 Emulator written in C++ with SDL2  

Based upon [Cowgod's Chip-8 Reference](http://devernay.free.fr/hacks/chip8/C8TECH10.HTM)  

## How to use

`./chip8 [path-to-rom]`
